Projeto para treinar as animações

Desenvolvi esse projeto como parte do meu aprendizado no DevClub, aplicando na prática tudo que venho estudando.

Trabalhei transições, movimentos, efeitos e pequenos detalhes que deixam a interface mais viva, também fui reforçando o uso do JavaScript para criar interações mais dinâmicas, deixando a página responder de forma mais natural ao que o usuário faz. No final, esse projeto virou um exercício prático que consolidou o que aprendi no DevClub: simples, eficiente e perfeito para evoluir minhas habilidades no front-end.
