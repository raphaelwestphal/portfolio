Projeto Tela de Cadastro

Esse projeto é uma tela de cadastro simples e objetiva, criada como exercício prático para treinar estrutura de formulários com foco em usabilidade.

A interface guia o usuário por um fluxo de preenchimento intuitivo: nome, email, senha, dados de endereço... tudo muito direto e com campos bem distribuídos, o que facilita o uso tanto em desktop quanto no celular. O botão de olho no campo é um toque a mais para simular interações reais de privacidade.
