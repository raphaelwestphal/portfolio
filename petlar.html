Projeto PetLar

Este projeto foi idealizado e desenvolvido por mim, com o objetivo de criar uma experiência completa para clínicas veterinárias, integrando tanto os atendimentos médicos quanto os cuidados de higiene do pet, como banho e tosa. A proposta do layout foi pensada para ser simples, funcional e intuitiva, permitindo que o usuário encontre as principais informações de forma rápida e organizada.

Na página inicial, desenvolvi um dashboard que reúne os dados mais relevantes para facilitar o acompanhamento do dia a dia, garantindo que tudo esteja sempre à mão. As demais telas seguem a mesma linha de praticidade, com uma interface clara e objetiva, evitando poluição visual e proporcionando uma navegação agradável.

O projeto ainda está em fase de desenvolvimento, mas já pode ser acompanhado em meu GitHub.
