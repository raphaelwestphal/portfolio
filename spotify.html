Projeto Interface do Spotify

Recriei a interface do Spotify, com foco na implementação dinâmica usando JavaScript.

Todas as seções de "Artistas Populares" e "Singles e Álbuns" foram criadas e inseridas dinamicamente via manipulação do DOM, demonstrando o uso de JavaScript para construção de interfaces interativas. Principais funcionalidades: Criação dinâmica de elementos com JavaScript, Manipulação e estilização do DOM, Organização de dados em seções.
